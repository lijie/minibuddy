# mini-buddy

一个极简架构的轻量级 AI 编程智能体，剖析、学习和复刻大厂 Agent 架构的教学项目. 使用 Rust 构建.

# 核心功能

* 多模型支持

    支持 Anthropic API 和 OpenAI 兼容接口，可自由切换 Claude、DeepSeek、Qwen、Kimi 等多种大模型.

* agent循环

    基于经典的 agent 循环架构，包含感知、思考、行动三个阶段，支持多轮交互和动态决策. 清晰展示大模型如何自主调用工具、思考（CoT）和反馈。支持如 Qwen 等模型的推理思考过程（reasoning_content）实时展示。

* Tool Use
    支持执行系统终端命令, 读取/写入本地文件.

* Bash沙盒

    实现了命令执行包装器的剥离与高危破坏性命令（如 rm -rf /）拦截。

* 支持 MCP

    支持模型上下文协议，实现工具的无缝扩充

* 支持 SKILL
* 炫酷的终端 UI
* 趣味彩蛋

* 流式输出

    支持逐 token 实时渲染到终端（SSE / chunked response），提供打字机效果的流畅体验.

* 上下文管理

    Token 计数与自动截断/摘要策略，防止对话超出模型上下文窗口. 对话历史持久化（SQLite 或 JSON），支持会话保存与恢复. 系统提示词管理，支持注入项目级上下文.

* 权限确认机制

    分级权限控制：读操作静默执行，写/删操作需用户确认. 支持白名单/黑名单配置，用户可自定义哪些命令自动放行. 展示大厂 Agent 如何平衡自主性与安全性.

* 重试与错误恢复

    API 超时、rate limit、网络断开的优雅处理与指数退避重试. Agent 循环中工具执行失败时的恢复策略.

* 配置系统

    API Key 管理（环境变量 / 配置文件），模型参数（temperature、max_tokens）可调. 多 profile 支持，快速切换不同模型组合.

* 代码 Diff 展示

    文件修改前后对比展示，类似 git diff 的终端着色渲染，让编程 Agent 的修改结果一目了然.

* 可观测性 / Debug 模式

    可视化展示每一轮 Agent 循环的完整过程：发送了什么 prompt、模型返回了什么、选择了哪个工具、工具返回了什么. 帮助学习者深入理解 Agent 内部运作机制.

# 架构分层

```
┌─────────────────────────────────┐
│           TUI Layer             │  ← Ratatui 渲染、输入处理、Markdown 着色
├─────────────────────────────────┤
│         Agent Loop              │  ← 感知→思考→行动 循环、CoT 展示
├─────────────────────────────────┤
│       Tool Registry             │  ← 内置工具 + MCP 工具统一接口
├─────────────────────────────────┤
│      LLM Provider Layer         │  ← 多模型适配器（Anthropic/OpenAI 兼容）
├─────────────────────────────────┤
│    Context / State Manager      │  ← 对话历史、token 管理、会话持久化
└─────────────────────────────────┘
```
